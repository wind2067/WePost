# 博群WePost
> 微博多账号批量发布工具
> 使用AI-Agent技术构建，代码完全开源，仅供学习使用，如有侵权，请私信
> 作者：wind_2067

## 使用教程
- 点击`server.bat`运行本地服务器环境
- 点击`WePost.html`打开网站
- 选中**账号管理**选项卡
  - 输入账户名用于后台管理，点击**添加**
  - 在对应账户的条目选择**登录**
- 选中**文案编辑**选项卡
  - 输入文字，选择配图
  - 点击**保存**以添加文案
  - 可以对文案进行**编辑**和**删除**
- 选中**发布配置**选项卡
  - 选择要发布的账号
  - 选择可见性、发布时间间隔(不含加载时间)、随机间隔
  - 点击**发布**，可预览内容，随机匹配文案
  - 点击以**确认发布**以发布内容，将跳转至**发布日志**
